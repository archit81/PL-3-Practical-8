Program to count digits in a number

Last updated : 01-01-2020

Write a code in any programming language that counts the digits in a given number.

Solution:

We can convert any number the number entered by user[1] into string and then find the length of the string to get the number of digits in the original number.
In the code below, count_digits will return the count of digits in the number 

#include <bits/stdc++.h> 
using namespace std; 
  
// To count the no. of digits in a number 
void count_digits(long long int n) 
{ 
    // converting number to string using 
    // to_string in C++ 
    string num = to_string(n); 
  
    // calculate the size of string 
    cout << num.size() << endl; 
} 
// driver function 
int main() 
{ 
    // number 
    long long int n; 
    cin >> n;
    count_digits(n); 
    return 0; 
}

[1] : This solution will only work for user input between -9223372036854775808 to 9223372036854775807 .

Copy the program in your IDE and compile and run (Press F11 in DevC++ OR F9 in CodeBlocks to Compile & execute) the program.

Sample Input : 
345289467
Sample Output : 
Number of digits : 9
Variables used :

n : Variable to store the number (base10) whose digit count is to be found.
count_digits : Function to calculate the number of digits in the passed parameter.

Time Complexity : 
O(N)
where N is the number of digits in given input n.
